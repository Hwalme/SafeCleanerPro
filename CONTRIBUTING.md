# Contributing to SafeCleanerPro

Thank you for your interest in contributing to SafeCleanerPro! We welcome contributions from everyone.

## Bug Reports and Feature Requests

Please use the [GitHub Issue Tracker](../../issues) to report bugs or request features. When filing an issue, please provide as much information as possible, including steps to reproduce the issue, OS version, and any relevant logs.

## Pull Requests

1. Fork the repository and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes.
5. Make sure your code follows our existing style guidelines.
6. Issue that pull request!

## Code Development

SafeCleanerPro is built using a modern stack. Please ensure you have the required SDKs installed for Windows development.

Thank you for making SafeCleanerPro better!
