# Security Policy

## Supported Versions

We actively support and patch the latest version of SafeCleanerPro. 

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of SafeCleanerPro seriously. If you find a security vulnerability, please do **not** open a public issue. Instead, report it responsibly.

To report a vulnerability, please email us at:
**security@safecleanerpro.org** (or contact the maintainer directly at femlanxuxu@gmail.com).

Please include the following in your report:
- A description of the vulnerability and its potential impact.
- Detailed steps to reproduce the vulnerability.
- Any suggested remediations.

We will acknowledge your report within 48 hours and work to provide a patch as quickly as possible.
